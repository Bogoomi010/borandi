# RANDI — 차원 균열 랜덤 디펜스 (재구축판)

Poly TD 스타일의 랜덤 디펜스. `docs/poly_td_video_analysis.md`의 분석을 기준으로
UI/연출/게임 흐름을 처음부터 재구축했다. 기존 프로젝트의 검증된 밸런스 데이터
(유닛/레시피/웨이브/난이도)는 그대로 이식했다.

## 실행

```bash
npm install
npm run dev      # http://localhost:5183
```

- `npm run build` — 타입체크 + 프로덕션 빌드
- `npm test` — vitest 엔진 테스트
- `node scripts/engine-smoke.cjs` — vitest 없이 도는 오프라인 엔진 스모크 (dist-test 필요)
- `npm run shots` — dev 서버 켠 상태에서 Playwright 스크린샷 플레이테스트

## 아키텍처

- **Pixi.js v8 단일 Application + 순수 TS** (React 없음). `SceneManager`가
  타이틀 → 스테이지 선택 → 전투 씬을 전환한다. 1280×720 고정 좌표계를
  창 크기에 맞춰 레터박스 스케일.
- **결정론 엔진 분리**: `src/core/engine.ts`는 렌더링 의존성이 전혀 없는
  20Hz 고정 틱 시뮬레이션. 시드 RNG(mulberry32)만 사용 → 같은 시드 = 같은 결과.
  렌더는 틱 사이를 보간해 60fps로 그린다.
- **이벤트 브리지**: 엔진이 `GameEvent[]`를 쌓고 UI가 프레임마다 drain하여
  로그/이펙트/데미지 숫자로 변환.

## 게임 규칙

- 소환(20G)으로 랜덤 유닛 획득 (일반 58 / 희귀 28 / 영웅 10 / 전설 4%).
  일반 10연속이면 다음 소환은 희귀 이상 확정.
- 같은 유닛 3개 = 다음 등급 랜덤 1개. 레시피 조합 = 지정 재료 + 골드.
  히든 등급은 레시피로만.
- 적은 순환로를 돌며 계속 쌓인다. **적 수가 난이도별 한계에 닿으면 패배.**
  라운드 시간이 다 되면 남은 적 위에 다음 웨이브가 겹쳐 온다.
- 10/20/30/40R 보스. 60초 내 못 잡으면 광폭화(잔당 소환). 보스 보상은
  등급 유닛 3택 1.
- 40R 최종 보스 클리어 = 승리. 다음 새 게임에서 다음 맵 선택권 해금
  (맵은 시작 시 선택, 1~40R 고정).
- **진영 강화(U)**: 골드로 화염 공격력 / 서리 감속 / 폭풍 공속 / 강철 보스딜 /
  공허 약화 / 숲 골드를 최대 8레벨까지 영구 강화 (해당 판 한정).

## 조작

| 키 | 동작 |
| --- | --- |
| S | 소환 |
| D | 선택 유닛 3개 조합 (미선택 시 가능한 첫 조합) |
| R | 레시피 도감 |
| U | 진영 강화 |
| X | 선택 유닛 판매 |

| 1 / 2 / 3 | 배속 |
| M | 음소거 |
| ESC | 일시정지 |
| 클릭 | 유닛 선택 / 선택 후 빈 칸 클릭 = 이동 |

## 구조

```
src/
  core/    rng, types, path(폐곡선 스플라인+슬롯 생성), engine(시뮬레이션)
  data/    units, recipes, waves, difficulty, upgrades (기존 밸런스 이식), stages(맵 5종)
  ui/      theme(토큰), widgets(메뉴식 다크 카드 버튼/게이지), tokens(유닛·적 벡터 토큰),
           RiftBackdrop('차원 균열' 테마: 암흑+금빛 글로우+다이아 파티클 — 전 씬 공용)
  audio/   sfx(절차 생성 WebAudio 효과음 — 오디오 에셋 0)
  scenes/  SceneManager, TitleScene, StageSelectScene, BattleScene
    battle/  BoardView(부유섬 전장+이펙트), HudViews(상단HUD/보스바/전력/로그),
             BottomPanel(유닛 상세+보유+액션), Popups(레시피/강화/보상/결과/일시정지)
  profile.ts  localStorage 해금/기록
```

## 디자인 원칙

- 이미지/오디오 에셋 0 — 모든 그래픽은 Pixi 벡터, 모든 소리는 WebAudio 신스.
- 기존 게임의 직접 그린 메뉴 화면 테마('차원 균열': 짙은 어둠 0x0c0f14 + 금빛 글로우 +
  떠오르는 진영색 다이아 파티클)를 전 씬에 통일. 전투 필드는 균열 위에 뜬 부유섬.
- 버튼은 메뉴 화면과 동일: 다크 카드(0x151b24) + 좌측 금색 액센트 바 + 호버 시 푸른 테두리.
- 콘솔 게임 문법으로 디자인: 모따기(chamfer) 프레임 + 금색 코너 브래킷, 앵귤러 명판(plaque)
  타이틀/HUD, 세그먼트 게이지, 룬 슬롯형 스킬 칩, 등급 글로우 초상 프레임, 키캡 프롬프트.
  웹 UI 요소(둥근 필, 플랫 카드, 네비바 느낌)는 사용하지 않는다.
